Static Pages release docs

heimq is a single-node Kafka-compatible broker with release-ready docs.

This microsite is intentionally plain HTML and CSS. It mirrors the checked-in README commands, the release workflows, and the repository's release-infrastructure plan without adding a package manager or build chain.

What ships here

Source, Docker, and Helm entry points are documented from one place so operators can move from first read to first run without hidden context.

Published pages come from .github/workflows/pages.yml on pushes to main.
